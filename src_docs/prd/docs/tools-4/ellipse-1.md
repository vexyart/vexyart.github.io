
![Ellipse tool](../media/doc360/Images/tool-mask-ellipse.svg) The **Ellipse** tool creates oval or circular masks that define where fills appear in your artwork.


### Activating the Ellipse Tool

Access the **Ellipse** tool via the toolbar or navigate to **Tools -> Ellipse** in the main menu.

![image.png](../media/doc360/Images/image-9JBDFWGV.png){width="360"}


Tool shortcut: {*O*}




### Using the Ellipse Tool

First, select a Layer or a fill. If you choose a fill, its Layer becomes active automatically. If nothing is selected, drawing will create a new Layer. Click and drag to create an elliptical mask, starting from your initial click point to where you release the mouse.

| mouse press | drag | release|
| --- | --- | --- |
|![image.png](../media/doc360/Images/image%28486%29.png){width="300"}|![image.png](../media/external/cdn.document360.io/image(487).png){width="300"}|![image.png](../media/external/cdn.document360.io/image(488).png){width="300"}|

> Press the {*⇧*} key while drawing to create a perfect circle.


### Smart Shape Detection
When you click on an object in your source image, the Ellipse tool will automatically detect and create an elliptical mask that covers that object:

- Click to detect a shape and replace the current mask
- {*⇧*} + Click to add another ellipse to the existing mask
- {*⌥*} + Click to subtract the detected elliptical area from the mask

### Erasing
Hold the {*⌥*} key and drag to erase elliptical portions from the mask.
