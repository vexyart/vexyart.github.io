![Freeform tool](../media/doc360/Images/tool-mask-freeform.svg) The **Freeform** tool lets you create masks with custom shapes by drawing freehand outlines.

### Activating the Freeform Tool

Access the Freeform tool by clicking its icon in the toolbar or selecting **Tools -> Freeform** from the main menu.
![image.png](../media/doc360/Images/image-4QCQ1N74.png){width="360"}

Tool shortcut: {*S*}

### Drawing with the Freeform Tool

Before using the Freeform tool, select a Layer or fill. Selecting a fill automatically activates its Layer. If nothing is selected, drawing will create a new Layer with your mask. To draw, press and hold the mouse button while dragging to create your shape. Release the mouse button to complete the mask on the selected Layer.

### Smart Shape Detection
When you click on an object in your source image, the Freeform tool will automatically detect and create a mask that matches the object's shape:

- Click to detect a shape and replace the current mask
- {*⇧*} + Click to add another shape to the existing mask
- {*⌥*} + Click to subtract the detected shape from the mask

### Erasing
Hold the {*⌥*} key and drag to erase portions of the mask.
