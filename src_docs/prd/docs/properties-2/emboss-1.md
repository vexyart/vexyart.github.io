
The **EMBOSS** effect gives your artwork depth by creating the appearance of raised or indented designs on the fill surface. This effect adds a 3D quality to your work, making it stand out with realistic lighting and shadows.

![image.png](../media/doc360/Images/image%28654%29.png){width="300"}

> Note: This effect is not applicable to Text, Trace, or Wireframe fills.


$~$

![](../media/external/cdn.document360.io/image1635(1).png){width="300"}

The emboss effect has two main properties that you can adjust to create exactly the look you want:

### Height

![Height](../media/doc360/Images/emboss-raise%282%29.svg) **Height** controls how pronounced the effect appears. Use positive values for a raised effect (making elements appear to come forward) or negative values for an indented effect (making elements appear to sink into the surface).

| height: 33% |  height: -33%  |  height: 100% |
| --- | --- | --- |
|![image.png](../media/doc360/Images/image%28636%29.png){width="300"}|![image.png](../media/external/cdn.document360.io/image(637).png){width="300"}|![image.png](../media/external/cdn.document360.io/image(638).png){width="300"}|


### Smoothness


![emboss-smooth.svg](../media/doc360/Images/emboss-smooth.svg) **Smoothness** adjusts how gradually the emboss effect transitions. Higher values create a softer, more blended effect, while lower values produce sharper, more defined edges.
