* **Auto Save**: Automatically saves the current document to prevent loss of work in case of unexpected interruptions or similar incidents. The document will be saved in a temporary file, which will be offered for recovery upon restarting the program.

* **Save Backup**: This feature ensures that when you save a document, the previous version of that document is also saved. The backup file will have the same name as the original but will have an *.~lines extension. This provides an added layer of safety, allowing you to revert to the previous version if needed.
