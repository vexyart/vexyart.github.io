The Mesh property of the layer provides an advanced level of flexibility by allowing you to manipulate fills using a grid-based system. This feature helps you create unique transformations and effects.

The following predefined mesh shapes are available:

![mesh-rect.svg](../media/doc360/Images/mesh-rect.svg)&nbsp;&nbsp;Rectangle
![mesh-donut.svg](../media/doc360/Images/mesh-donut.svg)&nbsp;&nbsp;Donut
![mesh-rising-wave.svg](../media/doc360/Images/mesh-rising-wave.svg)&nbsp;&nbsp;Rising wave
![mesh-falling-wave.svg](../media/doc360/Images/mesh-falling-wave.svg)&nbsp;&nbsp;Falling wave
![mesh-rising-ribbon.svg](../media/doc360/Images/mesh-rising-ribbon.svg)&nbsp;&nbsp;Rising ribbon
![mesh-falling-ribbon.svg](../media/doc360/Images/mesh-falling-ribbon.svg)&nbsp;&nbsp;Falling ribbon

*Note:* Meshes do not affect certain fill types such as trace, handmade, autowireframe, or scribble fills.

## Activating Mesh Mode

1. Select the layer you want to work with
2. Click the **Mesh** button in the Layers Panel toolbar
![image.png](../media/doc360/Images/image-PM5NWXZY.png){width="218"}
3. Choose one of the predefined mesh shapes
4. The mesh grid appears on your layer and a toggle appears in the layer row
![image.png](../media/doc360/Images/image-JER9NHUO.png){width="218"}

## Front and Back Sides

When a layer has Mesh mode enabled, fills can be assigned to different sides of the mesh:

![image.png](../media/doc360/Images/image-REXSML9H.png){width="218"}

- ![overlap-1.svg](../media/doc360/Images/overlap-1.svg) Fill appears on both sides
- ![overlap-2.svg](../media/doc360/Images/overlap-2.svg) Fill appears only on front side
- ![overlap-3.svg](../media/doc360/Images/overlap-3.svg) Fill appears only on back side

## Removing a Mesh

To remove a mesh from a layer:

1. Select the layer with the active mesh
2. Either:
   - Navigate to **Layer → Mesh → Remove**
   - Click the mesh icon in the layer row
![image.png](../media/doc360/Images/image-U4CIIXOL.png){width="215"}

> For detailed information about working with meshes, see the dedicated [Mesh chapter](/v1/docs/mesh-3).
