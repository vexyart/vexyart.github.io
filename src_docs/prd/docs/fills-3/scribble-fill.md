The **Scribble** fill type mimics the look of hand-drawn lines. It offers adjustable settings to control stroke spacing, curve shape, and line direction—allowing you to create an organic, hand-crafted effect.
  
![image.jpeg](../media/doc360/Images/image(179).png){width="400"}

## Fill Parameters

The following parameters let you fine-tune your scribble fill:

![image](../media/image-JlLgkW.png){width="300"}

![Interval](../media/doc360/Images/sm-interval.svg) **Interval** ([units](/v1/docs/units)): Adjusts the spacing between strokes. Lower values produce a denser pattern, while higher values create a more open design.

![Curviness](../media/doc360/Images/curviness.svg) **Curviness** (%): Controls the bend of the strokes. Higher percentages yield smoother, more fluid curves; lower percentages result in sharper, more angular turns.

![Randomization](../media/doc360/Images/radial-random.svg) **Randomization** (%): Introduces subtle variations to stroke placement for a more natural, less mechanical look.

![Angle](../media/doc360/Images/sm-angle.svg) **Angle** (°): Determines the overall orientation of the scribble strokes.

![Smoothness](../media/doc360/Images/wireframe-smooth.svg) **Smoothness**: Refines the flow of strokes by reducing jagged edges for a cleaner appearance.

![Details](../media/doc360/Images/wireframe-depth.svg) **Details** (%): Adjusts the complexity of the stroke pattern. Increase this value to add intricate lines or decrease it for a simpler design.

![scribble-rotation.svg](../media/doc360/Images/scribble-rotation%281%29.svg) **Rotation** (°): Applies incremental rotation to the scribble pattern, adding a dynamic twist to the design.

![scribble-length.svg](../media/doc360/Images/scribble-length.svg) **Height**: Controls the vertical amplitude of the scribble wave, affecting how tall the pattern appears.

![scribble-zigzag.svg](../media/doc360/Images/scribble-zigzag%281%29.svg) **Zigzag**: Displays the scribble as a continuous zigzag line.

![scribble-lines.svg](../media/doc360/Images/scribble-lines.svg) **Lines**: Presents the scribble as a series of linear strokes.

## Add and Customize a Scribble Fill

To create a new **Scribble** fill, follow the instructions in our [Add a Fill](vb://article/adding-a-fill-1). When the selection menu appears, choose the **Scribble** fill type.

![image.jpeg](../media/doc360/Images/image(180)-01.png){width="160"}

### Interval
1. In the **SCRIBBLE FILL** section, locate the **Interval** ![Interval](../media/doc360/Images/sm-interval.svg)  control.
2. Adjust the spacing between strokes using the slider or by entering a specific value.
3. Lower values create a denser pattern, while higher values yield a more open layout.

| interval: 0.6 | interval: 1 | interval: 1.5 |
| --- | --- | --- |
|![image.jpeg](../media/doc360/Images/image(182)-01.png){width="300"}|![image.jpeg](../media/external/cdn.document360.io/image(185).png){width="300"}|![image.jpeg](../media/external/cdn.document360.io/image(184).png){width="300"}|

### Curviness
1. Find the **Curviness** ![Curviness](../media/doc360/Images/curviness.svg) option.
2. Use the slider or enter a value manually to adjust the bend of the strokes.
3. Higher percentages yield smoother, flowing curves; lower percentages result in sharper angles.
   
| curviness: 0 | curviness: 50 | curviness: 100 |
| --- | --- | --- |

### Randomization
1. Locate the **Randomization** ![Randomization](../media/doc360/Images/radial-random.svg) setting.
2. Adjust the slider or enter a value to introduce slight variations in stroke placement.
3. Increasing this value creates a more natural, less uniform pattern.
