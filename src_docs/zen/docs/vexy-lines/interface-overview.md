Vexy Lines converts raster images into beautiful vector artwork. This guide introduces the main interface elements to help you start creating quickly.

![UM-Workspace-3.png](../media/doc360/Images/UM-Workspace-3%281%29.png){width="600"}

## Main Canvas

Your primary workspace for visualizing and editing artwork.

### Essential Navigation

Navigate your canvas efficiently with these shortcuts:

**Zooming**

* {*Z*}: Zoom in toward the cursor.
* {*X*}: Zoom out.
* {*⌘+*} / {*⌘−*} (Mac) or {*⌃+*} / {*⌃−*} (Windows): Quick zoom.
* {*⌘2*}: Zoom to the selected area.
* {*⌘3*}: Fit artwork to screen.
* {*⌥*} + **Mouse Wheel** or trackpad pinch: *Dynamic zoom adjustment*.

**Moving**

* {*␣*} + **Drag**: Pan across the canvas.
* {*⌘␣*}: Dynamic zoom or frame zoom on specific areas.

## Toolbar

Quick access to essential drawing and editing tools.

![vl-toolbar.png](../media/doc360/Images/vl-toolbar.png){width="600"}

### Editor Tools

![Editor tool](../media/doc360/Images/tool-edit.svg) **Editor Tool** {*V*}: Select, move, and modify objects and nodes.
![Meter tool](../media/doc360/Images/tool-meter%281%29.svg) **Meter Tool** {*R*}: Measure distances and angles.
![Pencil tool](../media/doc360/Images/tool-pencil%281%29.svg) **Pencil Tool** {*P*}: Draw freehand strokes.
![Knife tool](../media/doc360/Images/tool-knife.svg) **Knife Tool** {*K*}: Split and edit curves and paths.
![Transform tool](../media/doc360/Images/tool-transform%282%29.svg) **Transform Tool** {*⌘T*}/{*⌃T*}: Scale, rotate, and precisely adjust objects.

### Masking Tools

![Brush tool](../media/doc360/Images/tool-mask-brush.svg) **Brush Tool** {*B*}: Paint masks freehand with adjustable brush size.
![Rectangle tool](../media/doc360/Images/tool-mask-rectangle.svg) **Rectangle Tool** {*I*}: Create rectangular masks, including perfect squares.
![Ellipse tool](../media/doc360/Images/tool-mask-ellipse.svg) **Ellipse Tool** {*O*}: Draw circular or oval masks.
![Freeform tool](../media/doc360/Images/tool-mask-freeform.svg) **Freeform Tool** {*S*}: Create custom shapes or auto-detect complex areas.

### Refresh Options

![Auto Refresh](../media/doc360/Images/auto-refresh.svg) **Auto Refresh**: Automatically update artwork as you work.
![Refresh Fill](../media/doc360/Images/fill-refresh.svg) **Refresh Fill**: Quickly update the active fill.
![Refresh All](../media/doc360/Images/all-refresh.svg) **Refresh All**: Update all fills simultaneously.

### Undo & Redo

![undo](../media/doc360/Images/undo.svg) **Undo** {*⌘Z*}/{*⌘Z*}: Step backward through recent actions.
![redo](../media/doc360/Images/redo.svg) **Redo** {*⌘Z*}/{*⌘Z*}: Restore previously undone actions.

### Zoom Commands

![Zoom in](../media/doc360/Images/zoom-in.svg) **Zoom In**: Magnify the view for detailed editing.
![Zoom out](../media/doc360/Images/zoom-out.svg) **Zoom Out**: Reduce magnification to see more of your artwork.
![Zoom to Selected](../media/doc360/Images/zoom-selected.svg) **Zoom to Selected**: Frame selected objects automatically.
![Zoom to Actual Size](../media/doc360/Images/zoom-actual.svg) **Zoom to Actual Size**: Display artwork at 100% scale.

### View Controls

![Highlight edges](../media/doc360/Images/view-selection%286%29.svg) **Highlight Selection**: Emphasize edges of selected objects for easier editing.
![Show masks](../media/doc360/Images/view-controls%284%29.svg) **Highlight Masks**: Display mask boundaries clearly.
![Show fills](../media/doc360/Images/show-fills.svg) **Show Fills**: Toggle visibility of vector fills.
![Show images](../media/doc360/Images/show-images.svg) **Show Images**: Show or hide source images.

> Use commands in the **Window > Toolbar** menu to display Refresh, Undo, and Zoom buttons.

## Panels

Vexy Lines includes several panels for detailed control:

**Properties Panel**
Adjust fills, colors, masks, and effects.

**Layers Panel**
Organize layers for structured, editable documents.

**History Panel**
Track and manage recent actions.

**Preview Panel**
Instantly preview your artwork.

**Help Panel**
View contextual information about tools and features.

## Menu System

Access all features through logically grouped menus:

* **File**: Create, open, save, import, export
* **Edit**: Standard editing commands and preferences
* **View**: Workspace and viewing options
* **Layer**: Manage layers, groups, masks
* **Fill**: Create and modify vector fills
* **Tools**: Drawing and editing tools
* **Window**: Panel visibility and arrangement
* **Help**: Documentation, tutorials, Intro Tour

## Interactive Tour

Access the interactive Intro Tour anytime via **Help > Intro Tour**. This guided tour highlights key features, helping you quickly master the basics.
