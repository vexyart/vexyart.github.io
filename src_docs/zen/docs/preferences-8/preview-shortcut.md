The **Preview Shortcut** is a keyboard combination that activates a clean preview mode. In this mode, all control elements, source images, and highlights are hidden, displaying only the rendered result.

Using {*⌘ Space*}/{*⌃Space*}, you can choose between two zoom actions:

* **Dynamic Zoom:** Activated by pressing the key combination and dragging the mouse. Dragging to the right zooms in relative to the drag's starting point, while dragging to the left zooms out.
* **Frame Zoom:** Zooms into a specific area defined by drawing a frame with the mouse.
