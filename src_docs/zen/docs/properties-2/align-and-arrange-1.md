
The **ALIGN AND ARRANGE** section provides tools to precisely position and order your objects.


![image.png](../media/doc360/Images/image%28665%29.png){width="300"}

### Alignment

The alignment tools help you organize objects based on their bounding boxes. When multiple objects are selected, alignment is relative to the group's bounding box. With a single object, alignment is based on the document boundaries.


![Align Right](../media/doc360/Images/align-right.svg) **Align Right**: Aligns selected objects to the right edge of the bounding box.

![Align Center Horizontally](../media/doc360/Images/align-h-center.svg) **Align Center Horizontally**: Centers objects horizontally within the bounding box.

![Align Left](../media/doc360/Images/align-left%281%29.svg) **Align Left**: Aligns selected objects to the left edge of the bounding box.

![Align Top](../media/doc360/Images/align-top.svg) **Align Top**: Aligns selected objects to the top edge of the bounding box.

![Align Center Vertically](../media/doc360/Images/align-v-center.svg) **Align Center Vertically**: Centers objects vertically within the bounding box.

![Align Bottom](../media/doc360/Images/align-bottom.svg) **Align Bottom**: Aligns selected objects to the bottom edge of the bounding box.

### Order Adjustment

These commands control the stacking order of objects within a group, determining which objects appear in front of or behind others.


![Move object down one position](../media/doc360/Images/send-backward.svg) **Move Backward**: Shifts the selected object one position down in the stacking order.

![Move object up one position](../media/doc360/Images/bring-forward.svg) **Move Forward**: Shifts the selected object one position up in the stacking order.

![Make object the last in its group](../media/doc360/Images/send-back%281%29.svg) **Send to Back**: Places the selected object at the bottom of the stacking order.

![Make object the first in its group](../media/doc360/Images/bring-front.svg) **Bring to Front**: Places the selected object at the top of the stacking order.
