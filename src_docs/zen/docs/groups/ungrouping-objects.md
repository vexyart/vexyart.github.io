Sometimes you'll need to break a group apart to work with its individual components. 

**You might want to ungroup when:**

- You need to edit specific layers independently
- You're reorganizing your artwork structure  
- You want to move certain layers to different groups
- You need to apply different effects to individual layers
- You want to delete specific components without affecting others

Ungrouping doesn't delete anything—it simply releases the layers from their container so you can work with them separately.

### Ungrouping Objects

**To ungroup a selected group:**

1. Navigate to the menu **Layer -> Ungroup**.
2. Alternatively, use the shortcut {*⌘⇧G*}/{*⌃⇧G*}
