There are several ways to remove a layer from your document when you no longer need it.

### Using the Layers Panel

The quickest way to remove a layer is through the Layers panel:

1. Select the layer you want to remove by clicking on it in the Layers panel
2. Click the Delete button ![Delete object](../media/doc360/Images/trash.svg) at the bottom of the Layers panel

![image.png](../media/doc360/Images/image-OXGEQ57N.png){width="225"}

### Using Keyboard Shortcuts

You can also remove a selected layer using keyboard shortcuts:

- Press {*Del*} or {*Backspace*} key after selecting the layer

### Using the Menu

Another option is to use the menu:

1. Select the layer you want to remove
2. Go to **Layer > Delete**

> When you delete a layer, all fills and masks within that layer will also be deleted. This action cannot be undone after you save the document, so be careful when removing layers.

### What Happens When You Remove a Layer

When you remove a layer:

- All fills contained in that layer are permanently removed
- The layer's mask is deleted
- If the layer was in Mesh mode, all mesh settings are lost
- Any visual effects from that layer disappear from your artwork

### Tips for Managing Layers

Instead of deleting layers you're unsure about, consider:

- **Hiding layers** - Click the eye icon to hide a layer instead of deleting it
- **Locking layers** - Use the lock icon to prevent accidental changes
- **Creating groups** - Organize related layers into groups for better management
- **Renaming layers** - Give descriptive names to layers so you know their purpose

Sometimes it's better to hide a layer temporarily rather than delete it, especially when experimenting with different design approaches.
